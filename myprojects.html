Project-1

This application is designed in dialog flow, it plans user trip based on destination name given by the user. It provides different types of travel options to the user and ends with providing some basic accommodation details. This application uses intents, follow-up intents, contexts, webhook and fulfillment to drive a meaningful conversation with user. An intent in this application is derived based on previous assignments, in previous assignments user is provided with options for travel from Portland to New York JFK. But in this application source and destination can be any global city, it provides user car rental as a travel mode along with flights. It also asks user the type of car he prefers, it also provides the user possible hotel accommodations near destination airport.
