Project-1

This application is designed in Alexa skill, it plans trip to any city based on conversation with user. It provides hands-free option in planning a journey including auto reminders to user about travel. It provides different modes of travel options available between the cities, including car rentals and basic accommodation details. This application uses intents, follow-up intents, contexts, webhook and fulfillment to drive a meaningful conversation with user. This application helps user to plan their trip to global city, based on their preferences.

Project-2

The project is builds a DVD rental database. Tables in database are linked to each other. Schema of this database is designed in optimized way, so that query retrieval is faster and simpler. It comprises of multiple tables which stores Rental details, Customer details, Store details, staff details. An user interface is hosted on this database, which allows user to search for availability of DVD's in the store. The UI also allows customers to change their details or to search for nearby stores.

Project-3

This project is an simulation of 2048 grid game in single player format. It is designed based on AI algorithms, This has an advancement with available 2048 game, as it provides hints to the player in case of wrong steps. The bot is designed with winning as an estrablished goal. It is designed based on Minimax algorithms of AI.
